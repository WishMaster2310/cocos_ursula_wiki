# МодульОбнаружения

## Назначение

Модуль сканирует окружение в радиусе и сообщает HSM о найденных объектах, игроке, звуках и интерактивах.

## Идентификаторы из конфигурации

- Внутреннее имя: SensorModule
- Имя для HSM: МодульОбнаружения
- Иконка: 📡

## Входной API (actions)

### ПоискИгрокаВРадиусе

Запускает сканирование на наличие игрока в радиусе сенсора.

### ПоискОбъектаВРадиусеПоИмени

Ищет объект в радиусе по имени и обновляет данные о цели при нахождении.

### ПоискОбъектаВРадиусеПоИдентификатору

Ищет объект в радиусе по ID и обновляет данные о цели при нахождении.

### ВзаимодействиеИгрокаСОбъектом

Проверяет факт взаимодействия игрока с объектом и отправляет соответствующее событие.

### УстановитьРадиусОбнаружения

Изменяет рабочий радиус сканирования сенсора.

### ОстановкаПоиска

Останавливает активный цикл сканирования.

### ПоискЗвуковВРадиусе

Ищет звуковые источники в заданном радиусе слышимости.

## Выходной API (events)

### ИгрокОбнаружен

Событие отправляется, когда игрок найден в радиусе сенсора.

### ОбъектОбнаружен

Событие отправляется, когда найден требуемый объект.

### ИгрокНеОбнаружен

Событие отправляется, если игрок в радиусе не найден.

### ОбъектовНеОбнаружено

Событие отправляется, если подходящие объекты не найдены.

### ЗвукОбнаружен

Событие отправляется при обнаружении источника звука.

### ЗвукНеОбнаружен

Событие отправляется, если подходящий звук не найден.

### ВзаимодействиеИгрокаСОбъектом

Событие отправляется при подтвержденном взаимодействии игрока с текущим объектом.

### ВзаимодействиеИгрока

Событие отправляется как общий сигнал интеракции игрока.

## Параметры (params)

- РадиусОбнаружения

## Настройки (settings)

- DEFAULT_SENSOR_RADIUS
- DEFAULT_SENSOR_SCAN_INTERVAL
- MAX_SENSOR_RADIUS
- SCAN_CACHE_LIFETIME

## Примечание

Модуль часто выступает источником событий для переключения состояний patrol/chase/engage.
