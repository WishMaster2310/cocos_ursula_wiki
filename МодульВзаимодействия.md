# МодульВзаимодействия

## Назначение

Модуль управляет интерактивными действиями с объектом: проходимость, перемещение, уничтожение и запуск диалогов.

## Идентификаторы из конфигурации

- Внутреннее имя: InteractionModule
- Имя для HSM: МодульВзаимодействия
- Иконка: 🤝

## Входной API (actions)

### УстановитьПроходимость

Явно задает состояние проходимости объекта (проходимый или непроходимый).

### ПереключитьПроходимость

Инвертирует текущее состояние проходимости без необходимости знать его заранее.

### УничтожитьОбъект

Удаляет или деактивирует объект как интерактивную сущность сцены.

### ПереместитьОбъект

Перемещает объект в новые координаты или в указанную точку назначения.

### ПерейтиКДиалогу

Переводит взаимодействие к диалоговому сценарию или запускает соответствующий узел диалога.

## Выходной API (events)

### ОбъектУничтожен

Событие отправляется после успешного удаления или разрушения объекта.

### ОбъектПеремещён

Событие отправляется после завершения перемещения объекта.

### ПроходимостьИзменена

Событие отправляется при любом изменении состояния проходимости.

## Параметры (params)

- Проходимость

## Настройки (settings)

- INTERACTION_DISTANCE

## Примечание

Модуль удобно использовать в квестовой логике и динамических сценах со сменой проходимости.
