# МодульВзаимодействия

## Назначение

Модуль управляет интерактивными действиями с объектом: проходимость, перемещение, уничтожение и запуск диалогов.

## Идентификаторы из конфигурации

- Внутреннее имя: InteractionModule
- Имя для HSM: МодульВзаимодействия
- Иконка: 🤝

## Действия

### УстановитьПроходимость

Явно задает состояние проходимости объекта (проходимый или непроходимый).

```
УстановитьПроходимость()
```

### ПереключитьПроходимость

Инвертирует текущее состояние проходимости без необходимости знать его заранее.

```
ПереключитьПроходимость()
```

### УничтожитьОбъект

Удаляет или деактивирует объект как интерактивную сущность сцены.

```
УничтожитьОбъект()
```

### ПереместитьОбъект

Перемещает объект в новые координаты или в указанную точку назначения.

```
ПереместитьОбъект()
```

### ПерейтиКДиалогу

Переводит взаимодействие к диалоговому сценарию или запускает соответствующий узел диалога.

```
ПерейтиКДиалогу()
```

### УстановитьДистанциюВзаимодействия

Устанавливает максимальное расстояние, на котором возможно взаимодействие с объектом.

```
УстановитьДистанциюВзаимодействия(distance)
```

**Аргументы:**

1. `distance` (число) — дистанция взаимодействия в метрах (минимум 0.5)

**Пример:** `УстановитьДистанциюВзаимодействия(5)` — установит дистанцию взаимодействия 5 метров

## События

### ОбъектУничтожен

Событие отправляется после успешного удаления или разрушения объекта.

### ОбъектПеремещён

Событие отправляется после завершения перемещения объекта.

### ПроходимостьИзменена

Событие отправляется при любом изменении состояния проходимости.

## Переменные

- **Проходимость** — текущее состояние проходимости объекта (true/false)
- **ДистанцияВзаимодействия** — текущая дистанция взаимодействия в метрах

## Настройки (settings)

- **INTERACTION_DISTANCE** — максимальная дистанция для взаимодействия (по умолчанию: 3.0 метра)

## Примечание

- Модуль удобно использовать в квестовой логике и динамических сценах со сменой проходимости.
- Минимально допустимая дистанция взаимодействия составляет 0.5 метра. Значения меньше этого порога автоматически увеличиваются до минимума.
- Дистанция взаимодействия определяет радиус, в пределах которого игрок может взаимодействовать с объектом.
