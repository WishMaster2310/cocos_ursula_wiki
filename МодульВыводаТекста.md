# МодульВыводаТекста

## Назначение

Модуль показывает и скрывает текстовые блоки, привязанные к объекту, и сообщает HSM о состоянии отображения.

## Идентификаторы из конфигурации

- Внутреннее имя: TextModule
- Имя для HSM: МодульВыводаТекста
- Иконка: 📝

## Входной API (actions)

### ПоказатьТекст

Отображает текстовый блок с переданным содержимым и визуальными параметрами.

### СкрытьТекст

Скрывает ранее показанный текстовый блок.

## Выходной API (events)

### ТекстПоказан

Событие отправляется после успешного показа текста.

### ТекстСкрыт

Событие отправляется после скрытия текста.

## Параметры (params)

- ТекстВыводится

## Настройки (settings)

- FONT_SIZE
- TEXT_COLOR
- BACKGROUND_COLOR
- OFFSET_Y
- CONTENT_OFFSET
- LINE_HEIGHT
- BLOCK_WIDTH
- MAX_TEXT_LENGTH

## Примечание

Подходит для реплик NPC, подсказок и UI-подписей, управляемых состояниями HSM.
