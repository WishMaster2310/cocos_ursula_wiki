# МодульВыводаТекста

## Назначение

Модуль показывает и скрывает текстовые блоки, привязанные к объекту, и сообщает HSM о состоянии отображения.

## Идентификаторы из конфигурации

- Внутреннее имя: TextModule
- Имя для HSM: МодульВыводаТекста
- Иконка: 📝

## Действия

### ПоказатьТекст

Отображает текстовый блок с переданным содержимым и визуальными параметрами.

```
ПоказатьТекст(text)
```

**Аргументы:**

1. `text` (строка) — текст для отображения (максимальная длина: 400 символов)

**Пример:** `ПоказатьТекст("Привет, путник!")` — показать приветственное сообщение

### СкрытьТекст

Скрывает ранее показанный текстовый блок.

```
СкрытьТекст()
```

## События

### ТекстПоказан

Событие отправляется после успешного показа текста.

### ТекстСкрыт

Событие отправляется после скрытия текста.

## Переменные

- ТекстВыводится

## Настройки

- **FONT_SIZE** — размер шрифта (значение: 16)
- **TEXT_COLOR** — цвет текста (значение: "#ffffff")
- **BACKGROUND_COLOR** — цвет фона (значение: "#0000009d")
- **OFFSET_Y** — смещение по Y относительно объекта (значение: -0.4)
- **CONTENT_OFFSET** — отступ контента внутри блока (значение: 8)
- **LINE_HEIGHT** — межстрочный интервал относительно FONT_SIZE (значение: 1.32)
- **BLOCK_WIDTH** — ширина блока с текстом (значение: 320)
- **MAX_TEXT_LENGTH** — максимальная длина текста (значение: 400)

## Примечание

Подходит для реплик NPC, подсказок и UI-подписей, управляемых состояниями HSM.
