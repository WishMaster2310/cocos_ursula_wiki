# МодульДвижения

## Назначение

Модуль отвечает за перемещение объекта, обход препятствий и обновление данных о пройденной дистанции.

## Идентификаторы из конфигурации

- Внутреннее имя: MovementModule
- Имя для HSM: МодульДвижения
- Иконка: 🏃

## Входной API (actions)

### ДвигатьсяПоКоординатам

Запускает перемещение к указанным мировым координатам.

### ЗадатьКоординаты

Сохраняет целевые координаты без немедленного запуска движения.

### ДвигатьсяКОбъекту

Начинает движение к указанному объекту-цели с динамическим обновлением направления.

### ДвигатьсяОтОбъекта

Запускает движение от указанного объекта, увеличивая дистанцию до него.

### ЗадатьСлучайныеКоординаты

Выбирает случайную точку и устанавливает ее как цель перемещения.

### ЗадатьКоординатуВверх

Смещает цель движения вверх по оси карты.

### ЗадатьКоординатуВниз

Смещает цель движения вниз по оси карты.

### ЗадатьКоординатуВлево

Смещает цель движения влево по оси карты.

### ЗадатьКоординатуВправо

Смещает цель движения вправо по оси карты.

### СбросКоординат

Очищает текущую цель перемещения.

### ВернутьсяНаИсходную

Запускает движение к стартовой позиции объекта.

### Стоп

Прерывает текущее движение.

## Выходной API (events)

### ДвижениеНачато

Событие отправляется в момент начала перемещения.

### ПеремещениеВыполнено

Событие отправляется после достижения целевой точки.

### ДостигнутКрайКарты

Событие отправляется, когда объект упирается в ограничение карты.

### ПройденноеРасстояние

Периодическое событие с информацией о длине пути.

### СтолкновениеСПрепятствием

Событие отправляется при обнаружении препятствия на траектории.
