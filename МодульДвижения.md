# МодульДвижения

## Назначение

Модуль отвечает за перемещение объекта, обход препятствий и обновление данных о пройденной дистанции.

## Идентификаторы из конфигурации

- Внутреннее имя: MovementModule
- Имя для HSM: МодульДвижения
- Иконка: 🏃

## Действия

### ДвигатьсяПоКоординатам

Запускает перемещение к ранее заданным координатам.

```
ДвигатьсяПоКоординатам()
```

**Примечание:** Координаты должны быть предварительно установлены с помощью действия **ЗадатьКоординаты**.

### ЗадатьКоординаты

Сохраняет целевые координаты без немедленного запуска движения.

```
ЗадатьКоординаты(x, z)
```

**Аргументы:**

1. `x` (число) — координата X на карте
2. `z` (число) — координата Z на карте

**Пример:** `ЗадатьКоординаты(10, 15)` — задать целевую точку в координатах (10, 15)

### ДвигатьсяКОбъекту

Начинает движение к объекту-цели из АнализатораЦели с динамическим обновлением направления.

```
ДвигатьсяКОбъекту()
```

**Примечание:** Требует наличия активной цели в АнализатореЦели.

### ДвигатьсяОтОбъекта

Запускает движение от объекта-цели, увеличивая дистанцию до него.

```
ДвигатьсяОтОбъекта()
```

**Примечание:** Требует наличия активной цели в АнализатореЦели.

### ЗадатьСлучайныеКоординаты

Выбирает случайную точку в заданном радиусе и устанавливает ее как цель перемещения.

```
ЗадатьСлучайныеКоординаты(radius)
```

**Аргументы:**

1. `radius` (число, необязательно) — радиус случайного выбора точки в метрах (по умолчанию: 10)

**Пример:** `ЗадатьСлучайныеКоординаты(20)` — выбрать случайную точку в радиусе 20 метров

### ЗадатьКоординатуВверх

Смещает цель движения вверх по оси карты относительно текущей позиции объекта.

```
ЗадатьКоординатуВверх(distance)
```

**Аргументы:**

1. `distance` (число) — дистанция смещения в метрах

**Пример:** `ЗадатьКоординатуВверх(5)` — установить цель на 5 метров вверх от текущей позиции

### ЗадатьКоординатуВниз

Очищает текущую цель перемещения и останавливает движение.

```
СбросКоординат()
```

### ВернутьсяНаИсходную

Запускает движение к стартовой позиции объекта (позиция при создании объекта).

```
ВернутьсяНаИсходную()
```

### Стоп

Прерывает текущее движение.

```

```

## События

### ДвижениеНачато

Событие отправляется в момент начала перемещения.

### ПеремещениеВыполнено

Событие отправляется после достижения целевой точки.

### ДостигнутКрайКарты

Событие отправляется, когда объект упирается в ограничение карты.

### ПройденноеРасстояние

Периодическое событие с информацией о длине пути.

### СтолкновениеСПрепятствием

Событие отправляется при обнаружении препятствия на траектории.

## Переменные

- **ПройденноеРасстояние** — расстояние, пройденное за текущую сессию движения
- **ОбщееПройденноеРасстояние** — общее расстояние, пройденное объектом за все время

## Настройки

- **DEFAULT_MOVEMENT_SPEED** — скорость движения по умолчанию (значение: 1.75 м/с)
- **POSITION_EQUALITY_TOLERANCE** — допустимая погрешность при сравнении позиций (значение: 0.725 м)
- **TARGET_REACHED_DISTANCE** — минимальная дистанция, при которой объект считается достигнутым (значение: 1.2 м)
- **OBSTACLE_DETECTION_DISTANCE** — дистанция для обнаружения препятствий (значение: 5 м)
- **AVOIDANCE_DISTANCE** — дистанция обхода препятствий (значение: 4 м)
- **MAX_AVOIDANCE_DISTANCE** — максимальная дистанция обхода препятствий (значение: 10 м)
- **MAX_COLLECTED_CURRENT_DISTANCE** — максимальное расстояние, которое может быть накоплено за одну сессию движения (значение: 5000 м)
- **MAX_COLLECTED_DISTANCE** — максимальное общее расстояние, которое может быть накоплено (значение: 100000 м)
- **MOVEMENT_LOG_ENABLED** — включение логирования движения (значение: true)
- **MOVEMENT_COLLISION_SPEED** — коэффициент снижения скорости при столкновении (значение: 0.5)

## Примечание

Для поведения по целям обычно используется вместе с АнализаторЦели и МодульОбнаружения.
