# Счётчик

## Назначение

Модуль предоставляет один или несколько счетчиков для состояния объекта и дискретной логики в HSM.

## Идентификаторы из конфигурации

- Внутреннее имя: CounterModule
- Имя для HSM: Счётчик
- Иконка: 🔢

## Входной API (actions)

### ПрибавитьЗначение

Увеличивает значение выбранного счетчика на заданную величину.

### ОтнятьЗначение

Уменьшает значение выбранного счетчика на заданную величину.

### ОбнулитьЗначение

Сбрасывает значение выбранного счетчика в ноль.

## Выходной API (events)

### ЗначениеИзменилось

Событие отправляется при любом изменении значения счетчика.

## Параметры (params)

- ТекущееЗначение

## Настройки (settings)

- MAX_VALUE
- NUM_COUNTERS

## Примечание

Подходит для таймеров состояний, лимитов попыток, счетов и шагов сценария.
