# Счётчик

## Назначение

Модуль предоставляет один или несколько счетчиков для состояния объекта и дискретной логики в HSM.

## Идентификаторы из конфигурации

- Внутреннее имя: CounterModule
- Имя для HSM: Счётчик
- Иконка: 🔢

## Действия

### ПрибавитьЗначение

Увеличивает значение выбранного счетчика на заданную величину.

```
ПрибавитьЗначение(amount)
```

**Аргументы:**

1. `amount` (число, необязательно) — величина прибавления (по умолчанию 1)

**Пример:** `ПрибавитьЗначение(5)` — увеличить счётчик на 5

### ОтнятьЗначение

Уменьшает значение выбранного счетчика на заданную величину.

```
ОтнятьЗначение(amount)
```

**Аргументы:**

1. `amount` (число, необязательно) — величина уменьшения (по умолчанию 1)

**Пример:** `ОтнятьЗначение(3)` — уменьшить счётчик на 3

### ОбнулитьЗначение

Сбрасывает значение выбранного счетчика в ноль.

```
ОбнулитьЗначение()
```

## События

### ЗначениеИзменилось

Событие отправляется при любом изменении значения счетчика.

## Переменные

- ТекущееЗначение

## Настройки

- **MAX_VALUE** — максимальное значение счётчика (значение: 1000000)
- **NUM_COUNTERS** — количество счётчиков, доступных для одного объекта (значение: 2)

## Примечание

Подходит для таймеров состояний, лимитов попыток, счетов и шагов сценария.
