# МодульПовествования

## Назначение

Модуль интегрирует сценарную логику (узлы, переменные, наблюдение за значениями) в контур HSM.

## Идентификаторы из конфигурации

- Внутреннее имя: StoryModule
- Имя для HSM: МодульПовествования
- Иконка: 📖

## Входной API (actions)

### ПерейтиКУзлу

Принудительно переводит Story-граф на указанный узел. Используется для сценарных переходов из HSM, например после боя, триггера или взаимодействия.

### СледитьЗаПеременной

Включает наблюдение за указанной переменной сценария. Когда значение меняется, модуль может отправить событие в HSM для реактивного поведения.

### ПрекратитьСледитьЗаПеременной

Отключает ранее установленное наблюдение за переменной. Полезно, когда переход состояния больше не должен зависеть от этой переменной.

### ПерезапуститьСценарий

Сбрасывает текущий сценарный прогресс и запускает повествование заново. Удобно для повторных попыток, рестарта эпизода или отладки.

### ПолучитьЗначениеПеременной

Запрашивает текущее значение переменной Story для использования в логике состояний. Как правило, результат читается через параметры модуля.

### УстановитьИзПеременной

Копирует значение из глобального состояния (StateController) в переменную INK.

Аргументы:

- args[0] - имя переменной состояния
- args[1] - имя переменной в INK (опционально; если не указано, используется args[0])

## Выходной API (events)

### ПовествованиеЗавершено

Событие отправляется, когда сценарий дошел до финала или точки завершения. По нему HSM обычно переводит объект в итоговое состояние.

### ЗначениеПеременнойИзменилось

Событие отправляется при изменении наблюдаемой переменной Story. По нему HSM может выполнять условные переходы, включая/выключая поведение в реальном времени.

## Параметры (params)

- ТекущийУзел
- ПовествованиеЗавершено
- ОтслеживаемоеЗначение
- НазваниеОтслеживаемойПеременной

- SHOW_DIALOGS_LABEL
- HIDE_DIALOGS_LABEL
- PANEL_WIDTH
- SET_VARIABLE_FUNCTION_NAME

## Примечание

Используется для связки нарратива и поведения объектов без жесткой привязки к конкретным состояниям.
