# Триггер

## Назначение

Модуль отправляет триггерное событие объектам в радиусе, чтобы они могли отреагировать через свои HSM.

## Идентификаторы из конфигурации

- Внутреннее имя: TriggerModule
- Имя для HSM: Триггер
- Иконка: 🚪

## Входной API (actions)

### ТриггерВРадиусе

Активирует триггер с указанным радиусом: модуль проверяет объекты вокруг центра и уведомляет попавшие в радиус цели.

## Выходной API (events)

### ТриггерАктивированВРадиусе

Событие отправляется объекту, который попал в радиус активированного триггера.

## Параметры (params)

- Нет публичных параметров

## Настройки (settings)

- Нет обязательных настроек в конфиге

## Примечание

Для корректной работы вычислений радиуса используйте мировые координаты (world position) центра триггера.
