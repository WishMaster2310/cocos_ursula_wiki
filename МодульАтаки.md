# МодульАтаки

## Назначение

Модуль управляет атакой текущей цели: проверяет дистанцию, выполняет удар и сообщает HSM о результате.

## Идентификаторы из конфигурации

- Внутреннее имя: AttackModule
- Имя для HSM: МодульАтаки
- Иконка: ⚔️

## Входной API (actions)

### АтаковатьЦель

Запускает попытку атаки текущей цели с учетом дистанции, шанса попадания, кулдауна и параметров урона.

### Стоп

Останавливает текущий атакующий цикл и прекращает новые попытки удара.

## Выходной API (events)

### АтакаВыполнена

Событие отправляется после успешной атаки и применения урона.

### ЦельВошлаВЗону

Событие отправляется, когда цель оказалась в пределах дистанции атаки.

### ЦельВышлаИзЗоны

Событие отправляется, когда цель вышла за пределы дистанции атаки.

## Параметры (params)

- ЦельВЗоне
- УронЗаАтаку

## Настройки (settings)

- RANGE
- DAMAGE
- HIT_COOLDOWN
- HIT_CHANCE
- CRIT_CHANCE
- CRIT_DAMAGE

## Примечание

Используйте названия действий, событий и параметров в HSM строго в кириллическом виде из config.modules.AttackModule.
