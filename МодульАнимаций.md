# МодульАнимаций

## Назначение

Модуль управляет воспроизведением анимаций и отправляет события об их завершении.

## Идентификаторы из конфигурации

- Внутреннее имя: AnimationModule
- Имя для HSM: МодульАнимаций
- Иконка: 🎬

## Действия

### ВоспроизвестиАнимацию

Запускает конкретную анимацию по имени с заданными параметрами воспроизведения.

```
ВоспроизвестиАнимацию(animationName, repeatCount)
```

**Аргументы:**

1. `animationName` (строка) — имя анимации для воспроизведения
2. `repeatCount` (число, необязательно) — количество повторений (0 = зациклено, по умолчанию 1)

**Пример:** `ВоспроизвестиАнимацию("walk", 3)` — воспроизвести анимацию "walk" 3 раза

### ОстановитьАнимацию

Прерывает текущую анимацию или анимационный цикл.

```
ОстановитьАнимацию()
```

### ВоспроизвестиСлучайнуюАнимацию

Случайно выбирает анимацию из набора и запускает ее, что удобно для вариативных idle/ambient состояний.

```
ВоспроизвестиСлучайнуюАнимацию(repeatCount)
```

**Аргументы:**

1. `repeatCount` (число, необязательно) — количество повторений (0 = зациклено, по умолчанию 1)

**Пример:** `ВоспроизвестиСлучайнуюАнимацию(0)` — зациклить случайную анимацию

## События

### ЦиклАнимацийЗавершился

Событие отправляется при завершении одного цикла анимации, полезно для повторяемых паттернов.

### АнимацияЗавершилась

Событие отправляется, когда анимация полностью завершена и больше не воспроизводится.

## Переменные

- ТекущаяАнимация
- СписокАнимаций
- КоличествоАнимаций
- ТекущийНомерЦикла

## Настройки

- **DEFAULT_ANIMATION_REPEAT_COUNT** — количество повторений анимации по умолчанию (значение: 1; 0 = зациклено)

## Примечание

Подходит для синхронизации боевых окон, idle/walk/attack циклов и сценарных сцен.
