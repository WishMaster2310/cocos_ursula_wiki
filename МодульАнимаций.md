# МодульАнимаций

## Назначение

Модуль управляет воспроизведением анимаций и отправляет события об их завершении.

## Идентификаторы из конфигурации

- Внутреннее имя: AnimationModule
- Имя для HSM: МодульАнимаций
- Иконка: 🎬

## Входной API (actions)

### ВоспроизвестиАнимацию

Запускает конкретную анимацию по имени с заданными параметрами воспроизведения.

### ОстановитьАнимацию

Прерывает текущую анимацию или анимационный цикл.

### ВоспроизвестиСлучайнуюАнимацию

Случайно выбирает анимацию из набора и запускает ее, что удобно для вариативных idle/ambient состояний.

## Выходной API (events)

### ЦиклАнимацийЗавершился

Событие отправляется при завершении одного цикла анимации, полезно для повторяемых паттернов.

### АнимацияЗавершилась

Событие отправляется, когда анимация полностью завершена и больше не воспроизводится.

## Параметры (params)

- ТекущаяАнимация
- СписокАнимаций
- КоличествоАнимаций
- ТекущийНомерЦикла

## Настройки (settings)

- DEFAULT_ANIMATION_REPEAT_COUNT

## Примечание

Подходит для синхронизации боевых окон, idle/walk/attack циклов и сценарных сцен.
