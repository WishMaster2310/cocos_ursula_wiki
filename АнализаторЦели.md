# АнализаторЦели

## Назначение

Модуль хранит текущую цель, отслеживает ее состояние и дистанцию, а также сообщает HSM о потере цели.

## Идентификаторы из конфигурации

- Внутреннее имя: TargetAnalyzer
- Имя для HSM: АнализаторЦели
- Иконка: 🎯

## Входной API (actions)

### СброситьЦель

Сбрасывает текущую цель анализатора. Используется, когда объект должен выйти из режима преследования или начать поиск новой цели.

## Выходной API (events)

### ЦельПотеряна

Событие отправляется, когда цель вышла за допустимую дистанцию, была удалена или перестала быть валидной.

### ЗдоровьеЦелиИсчерпано

Событие отправляется, когда у цели здоровье достигло нуля. Обычно по нему HSM переключает поведение на следующую цель или завершение боя.

## Параметры (params)

- ПозицияЦели
- РасстояниеДоЦели
- ЗдоровьеЦели
- ЗдоровьеЦелиПроц
- ЗдоровьеЦелиИсчерпано
- УронЦелиЗаАтаку

## Настройки (settings)

- UPDATE_RATE
- MAX_DISTANCE

## Примечание

Используется как центральный источник данных о текущей цели для боевых и поведенческих модулей.
