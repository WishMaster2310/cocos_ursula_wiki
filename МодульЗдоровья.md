# МодульЗдоровья

## Назначение

Модуль хранит и изменяет здоровье объекта, а также уведомляет HSM об изменениях и обнулении.

## Идентификаторы из конфигурации

- Внутреннее имя: HealthModule
- Имя для HSM: МодульЗдоровья
- Иконка: ❤️

## Действия

### ПрибавитьЗначение

Увеличивает текущее здоровье на заданную величину.

```
ПрибавитьЗначение(amount)
```

**Аргументы:**

1. `amount` (число) — величина прибавления здоровья

**Пример:** `ПрибавитьЗначение(10)` — восстановить 10 единиц здоровья

### ОтнятьЗначение

Уменьшает текущее здоровье на заданную величину.

```
ОтнятьЗначение(amount)
```

**Аргументы:**

1. `amount` (число) — величина урона здоровью

**Пример:** `ОтнятьЗначение(5)` — нанести 5 единиц урона

### УстановитьЗначение

Присваивает здоровью точное значение.

```
УстановитьЗначение(value)
```

**Аргументы:**

1. `value` (число) — новое значение здоровья

**Пример:** `УстановитьЗначение(100)` — установить здоровье равным 100

## События

### ЗначениеИзменилось

Событие отправляется при любом изменении здоровья.

### ЗдоровьеИсчерпано

Событие отправляется, когда здоровье достигает нуля.

## Переменные

- ТекущееЗначение
- ТекущееЗначениеПроц

## Настройки

- **DEFAULT_HP** — здоровье по умолчанию (значение: Infinity = бесконечность)

## Примечание

Модуль обычно используется вместе с МодульАтаки и АнализаторЦели для боевой логики.
