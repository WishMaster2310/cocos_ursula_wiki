# МодульЗдоровья

## Назначение

Модуль хранит и изменяет здоровье объекта, а также уведомляет HSM об изменениях и обнулении.

## Идентификаторы из конфигурации

- Внутреннее имя: HealthModule
- Имя для HSM: МодульЗдоровья
- Иконка: ❤️

## Входной API (actions)

### ПрибавитьЗначение

Увеличивает текущее здоровье на заданную величину.

### ОтнятьЗначение

Уменьшает текущее здоровье на заданную величину.

### УстановитьЗначение

Присваивает здоровью точное значение.

## Выходной API (events)

### ЗначениеИзменилось

Событие отправляется при любом изменении здоровья.

### ЗдоровьеИсчерпано

Событие отправляется, когда здоровье достигает нуля.

## Параметры (params)

- ТекущееЗначение
- ТекущееЗначениеПроц

## Настройки (settings)

- DEFAULT_HP

## Примечание

Модуль обычно используется вместе с МодульАтаки и АнализаторЦели для боевой логики.
