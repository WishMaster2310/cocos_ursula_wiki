# ВоспроизведениеЗвука

## Назначение

Модуль управляет проигрыванием звуков, громкостью и радиусом слышимости, отправляя события о состоянии аудио.

## Идентификаторы из конфигурации

- Внутреннее имя: SoundModule
- Имя для HSM: ВоспроизведениеЗвука
- Иконка: 🔊

## Действия

### ВоспроизвестиЗвук

Запускает воспроизведение указанного звука. Можно использовать для одноразовых эффектов и фоновых реплик.

```
ВоспроизвестиЗвук(soundName, hearingRadius)
```

**Аргументы:**

1. `soundName` (строка) — имя звукового файла для воспроизведения
2. `hearingRadius` (число, необязательно) — радиус слышимости звука в метрах (если не указан, используется текущий радиус)

**Пример:** `ВоспроизвестиЗвук("door_open", 20)` — воспроизвести звук "door_open" с радиусом слышимости 20 метров

### Стоп

Останавливает текущее воспроизведение указанного звука.

```
Стоп(soundName)
```

**Аргументы:**

1. `soundName` (строка, необязательно) — имя звука для остановки (если не указан, останавливается активный звук)

### ОстановитьВсеЗвуки

Принудительно останавливает все звуки, которыми управляет модуль на объекте.

```
ОстановитьВсеЗвуки()
```

### УстановитьГромкость

Изменяет текущую громкость воспроизведения.

```
УстановитьГромкость(volume)
```

**Аргументы:**

1. `volume` (число) — громкость от 0.0 до 1.0 (0.0 = тишина, 1.0 = максимальная громкость)

**Пример:** `УстановитьГромкость(0.5)` — установить громкость на 50%

### УстановитьРадиусСлышимости

Задает дистанцию, на которой звук считается слышимым для сенсоров и слушателей.

```
УстановитьРадиусСлышимости(radius)
```

**Аргументы:**

1. `radius` (число) — радиус слышимости в метрах

**Пример:** `УстановитьРадиусСлышимости(100)` — установить радиус слышимости 100 метров

### ВоспроизвестиСлучайныйЗвук

Выбирает случайный звук из доступного набора и запускает его воспроизведение.

```
ВоспроизвестиСлучайныйЗвук()
```

## События

### ЗвукВоспроизведён

Событие отправляется после успешного запуска воспроизведения.

### ЗвукОстановлен

Событие отправляется после остановки звука вручную или через команду стопа.

### ЗвукЗавершён

Событие отправляется, когда звук завершился естественным образом.

## Переменные

- ТекущаяГромкость
- Воспроизводится
- РадиусСлышимости
- КоличествоЗвуков

## Настройки

- **DEFAULT_HEARING_RADIUS** — радиус слышимости по умолчанию (значение: 1000 м)
- **DEFAULT_VOLUME** — громкость по умолчанию (значение: 1.0 = 100%)

## Примечание

Для реакций NPC на звук обычно комбинируется с МодульОбнаружения и АнализаторЦели.
