# МодульИндикации

## Назначение

Модуль управляет визуальной индикацией (показ/скрытие/цвет) и сообщает HSM о смене состояния индикатора.

## Идентификаторы из конфигурации

- Внутреннее имя: IndicatorModule
- Имя для HSM: МодульИндикации
- Иконка: 💡

## Входной API (actions)

### ВключитьИндикатор

Показывает индикатор на объекте.

### ВыключитьИндикатор

Скрывает индикатор на объекте.

### УстановитьЦветИндикатора

Задает индикатору конкретный цвет.

### ПоказатьСЦветом

Одновременно устанавливает цвет и показывает индикатор.

### УстановитьСлучайныйЦвет

Выбирает случайный цвет индикатора.

## Выходной API (events)

### ИндикаторВключен

Событие отправляется после успешного показа индикатора.

### ИндикаторВыключен

Событие отправляется после скрытия индикатора.

### ЦветИндикатораИзменён

Событие отправляется после изменения цвета.

## Параметры (params)

- ВидимостьИндикатора
- ТекущийЦвет

## Настройки (settings)

- DEFAULT_INDICATOR_DURATION
- MAX_INDICATOR_DURATION

## Примечание

Удобен для сигнализации состояний AI и отладочных маркеров объектов.
