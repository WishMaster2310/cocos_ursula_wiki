# МодульИндикации

## Назначение

Модуль управляет визуальной индикацией (показ/скрытие/цвет) и сообщает HSM о смене состояния индикатора.

## Идентификаторы из конфигурации

- Внутреннее имя: IndicatorModule
- Имя для HSM: МодульИндикации
- Иконка: 💡

## Действия

### ВключитьИндикатор

Показывает индикатор на объекте.

```
ВключитьИндикатор()
```

### ВыключитьИндикатор

Скрывает индикатор на объекте.

```
ВыключитьИндикатор()
```

### УстановитьЦветИндикатора

Задает индикатору конкретный цвет.

```
УстановитьЦветИндикатора(color)
```

**Аргументы:**

1. `color` (строка) — цвет индикатора ("red"/"красный", "green"/"зелёный", "blue"/"синий")

**Пример:** `УстановитьЦветИндикатора("зелёный")` — установить зелёный цвет индикатора

### ПоказатьСЦветом

Одновременно устанавливает цвет и показывает индикатор.

```
ПоказатьСЦветом(color)
```

**Аргументы:**

1. `color` (строка) — цвет индикатора ("red"/"красный", "green"/"зелёный", "blue"/"синий")

**Пример:** `ПоказатьСЦветом("red")` — показать красный индикатор

### УстановитьСлучайныйЦвет

Выбирает случайный цвет индикатора.

```
УстановитьСлучайныйЦвет()
```

## События

### ИндикаторВключен

Событие отправляется после успешного показа индикатора.

### ИндикаторВыключен

Событие отправляется после скрытия индикатора.

### ЦветИндикатораИзменён

Событие отправляется после изменения цвета.

## Переменные

- ВидимостьИндикатора
- ТекущийЦвет

## Настройки

- **DEFAULT_INDICATOR_DURATION** — длительность отображения индикатора по умолчанию (значение: 2.0 секунды)
- **MAX_INDICATOR_DURATION** — максимальная длительность отображения индикатора (значение: 10.0 секунд)

## Примечание

Удобен для сигнализации состояний AI и отладочных маркеров объектов.
