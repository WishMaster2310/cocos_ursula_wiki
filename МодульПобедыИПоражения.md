# МодульПобедыИПоражения

## Назначение

Модуль формирует сигналы победы и поражения и предоставляет централизованную точку завершения игровой сессии.

## Идентификаторы из конфигурации

- Внутреннее имя: WinLooseModule
- Имя для HSM: МодульПобедыИПоражения
- Иконка: 🏆

## Действия

### Победа

Фиксирует победное завершение и запускает соответствующую реакцию сценария/UI.

```
Победа()
```

### Поражение

Фиксирует проигрыш и запускает соответствующую реакцию сценария/UI.

```
Поражение()
```

## События

### ИгрокПобедил

Событие отправляется после фиксации победы.

### ИгрокПроиграл

Событие отправляется после фиксации поражения.

## Переменные

- Нет публичных переменных

## Настройки

- **WIN_TEXT** — текст победы (значение: "Вы победили!")
- **LOOSE_TEXT** — текст поражения (значение: "Вы проиграли!")

## Примечание

Модуль удобен как единый финальный шлюз для любых win/lose-веток в HSM.
